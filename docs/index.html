Z80emulator

Pythonベースの Z80 CPU エミュレータ & BASIC-to-Z80 コンパイラ

  • BASIC 文法
  • アセンブラ文法
  • Z80 命令セット

特徴

  • 完全な Z80 命令セットをサポート（基本 + CB + ED + DD/FD プレフィックス）
  • Visual Basic 風の簡易命令を Z80 マシン語に変換
  • CPU、メモリ、フラグをモジュールとして分離
  • ラベル・条件ジャンプ対応
  • PyPI 配布済みで簡単インストール

インストール

pip install Z80emulator

使い方の概要

BASIC 風プログラムやアセンブラコードをコンパイルして Z80 CPU 上で実行できます。詳細は以下のページを参照してください：

  • BASIC 文法と使用例
  • アセンブラ文法と使用例
  • Z80 命令セット
